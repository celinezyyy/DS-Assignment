# Data Structure Assignment - Flight Ticket Booking System

**Project Description:**

This project developed a Flight Ticket Booking System that allows users to search, book, edit, view, and cancel flight tickets, including managing a waiting list queue. This system ensures efficient ticket management and automatically handles confirmed and waiting list passengers.

**Key Features:**

1. Flight Search: Users can search for flights in any week of their choice.

2. Ticket Booking: Books a seat if available; otherwise, adds the passenger to the waiting list.

3. Edit Ticket Information: Allows updating passenger details such as name or passport number.

4. Ticket Status: View confirmed or waiting list status.

5. Ticket Cancellation: Automatically updates waiting list, promoting the first passenger to a confirmed seat when a ticket is canceled.

_**Technologies:**_ Java 

_**Focus:**_ Backend Logic · Queue · CRUD Operations · Array
